# HH.ru PHP Vacancy Fetcher

Программа для получения списка вакансий PHP с сайта hh.ru через публичный API.

## Требования

- Go 1.21 или выше

## Использование

### Запуск без компиляции:
```bash
go run main.go
```

### Компиляция и запуск:
```bash
go build -o hh-vacancies main.go
./hh-vacancies
```

## Результат

Программа выводит список вакансий в формате:
- Название вакансии
- Компания
- Ссылка на вакансию

По умолчанию получает первые 100 вакансий из России.

## Настройка

В файле `main.go` можно изменить:

- `searchText` - поисковый запрос (по умолчанию "PHP")
- `area` в параметрах API - регион поиска (113 - Россия)
- Раскомментировать цикл для получения нескольких страниц результатов

## API

Программа использует публичный API hh.ru: https://api.hh.ru/vacancies
